# Mario_multiverseV1

📖 About the Project
Super Multiverse is a fan-made 2D platformer built with Python and Pygame. The player controls Mario in a procedurally generated infinite world where four iconic universes collide. Jump across Mushroom Kingdom platforms, battle Stormtroopers with a lightsaber, catch Pokémon companions, and face Thanos wielding Infinity Stones — all in one seamless adventure.
